# 🌍✨ Earthquake Prediction Project 🌟

Welcome to the Earthquake Prediction Project! 

This project utilizes Machine Learning to predict earthquakes and provides users with real-time predictions through a sleek, responsive web interface.

## Developed using:

* Frontend: ReactJS + Tailwind CSS 🖥️
* Backend: Django REST Framework 🛠️
  
# 🎨 Project Highlights

* ✨ Real-time Predictions
* 🌐 Interactive Web Interface built with ReactJS & Tailwind CSS
* 📊 Data Visualizations: Graphs and trends in earthquake activity
* 🔍 Seismic Analysis powered by Machine Learning algorithms
* ⚙️ Backend API built with Django REST Framework

# 🛠️ Tech Stack
### Frontend 🖥️
* ReactJS ⚛️: For dynamic UI
* Tailwind CSS 🎨: For modern, responsive design
### Backend 🔧
* Django 🐍: Python-based web framework
* Django REST Framework 🔗: To handle API endpoints
* Machine Learning 🤖
* Jupyter Notebook 📒: For training ML models
* Scikit-learn 📈: For building the earthquake prediction model
* Pandas 🐼 & NumPy 🧮: Data manipulation and analysis
  
# 🌟 How It Works
* Data Collection 📊: Seismic data is gathered from the Kaggle Earthquake Dataset.
* Model Training 🤖: The machine learning model is trained to predict future seismic activities.
* Prediction 🌍: The model makes real-time earthquake predictions based on input data.
* Visualization 🖼️: Users can view earthquake trends and forecasts on the website's dashboard.
